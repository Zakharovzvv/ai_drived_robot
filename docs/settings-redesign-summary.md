# Settings Page Redesign Summary

## Выполненные изменения

### Структура настроек

Страница настроек полностью переработана с разделением по устройствам:

1. **Connection to ESP32** (Подключение к ESP32)
   - Объединяет Control Transport Mode и Wi-Fi Network Configuration
   - Устраняет дублирование между Control Link и Wi-Fi Settings
   - Две подсекции с визуальным разделением:
     - Control Transport Mode: выбор автоматического/ручного режима (Wi-Fi/UART)
     - Wi-Fi Network Configuration: MAC адрес, IP, порт, путь WebSocket

2. **Camera Settings** (Настройки камеры)
   - Независимая секция для настройки разрешения и качества JPEG
   - Список доступных разрешений с указанием поддержки firmware

3. **Shelf Map** (Карта полок)
   - Отдельная секция для конфигурации 3×3 цветовой сетки
   - Возможность сохранения в NVS

4. **ESP32 Settings** (Настройки ESP32)
   - Placeholder для будущих системных параметров

5. **Arduino UNO Settings** (Настройки Arduino UNO)
   - Placeholder для калибровки сенсоров и параметров движения

### Порядок секций на странице

```text
System Settings
├── Connection to ESP32
│   ├── Control Transport Mode
│   └── Wi-Fi Network Configuration
├── Camera Settings
├── Shelf Map
├── ESP32 Settings
└── Arduino UNO Settings
```

### Преимущества новой структуры

1. **Устранение дублирования**: Control Link и Wi-Fi Settings объединены в одну логическую группу
2. **Ясная группировка**: настройки сгруппированы по устройствам, а не по функциям
3. **Масштабируемость**: легко добавить новые настройки для ESP32 или UNO
4. **Визуальная иерархия**: subsections с отступами и разделителями улучшают читаемость
5. **Консистентность**: единый стиль для всех секций
